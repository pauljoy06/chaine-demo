# Chaine Demo

## Tech Stack
- Node18
- Chakra UI
- [Create React App](https://github.com/facebook/create-react-app).

## Steps to run the Frontend
### Step 1
Make sure Node18 is installed.
Follow the instructions [here](https://nodejs.org/en/download) if node is not installed

### Step 2
Run `npm install`

### Step 3
Run `npm start`
